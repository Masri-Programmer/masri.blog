# 🎨 Frontend Resources

A curated list of resources to help you build beautiful, performant, and accessible user interfaces.

## 🎨 Design & Prototyping
- [Penpot](https://penpot.app/) - Open-source design and prototyping platform.
- [Balsamiq](https://balsamiq.com/) - Rapid wireframing tool for quick mockups.
- [Nerd Fonts](https://www.nerdfonts.com/font-downloads) - Patched fonts for developers (icons/glyphs).
- [Color Adobe Wheel](https://color.adobe.com/create/color-wheel) - Tool for choosing color schemes.
- [Dribbble](https://dribbble.com/) - Design inspiration and community.

## 💻 Frameworks & UI Kits
- [Vue.js](https://vuejs.org/) - The progressive JavaScript framework.
- [Blueprint](https://blueprintjs.com/) - React UI toolkit for data-dense interfaces.
- [Evergreen](https://evergreen.segment.com/components/alert) - React UI framework by Segment.
- [Flowbite](https://flowbite.com/) - UI components built with Tailwind CSS.
- [Shadcn for Vue](https://www.shadcn-vue.com/) - Beautifully designed components for Vue.
- [Tailbits](https://tailwind-generator.com/generators?ref=dailydev) - Customizable Tailwind CSS components.

## 🛠️ Development Tools
- [Create Next App](https://create-next-app.vercel.app/) - Bootstrap Next.js apps quickly.
- [Vite](https://vitejs.dev/) - Next generation frontend tooling.
- [Frontend Dev Bookmarks](https://github.com/dypsilon/frontend-dev-bookmarks) - Comprehensive list for frontend devs.
- [cdnjs](https://cdnjs.com/) - Free CDN for JavaScript libraries.

## 🎞️ Animation & Charts
- [anime.js](https://animejs.com/) - Lightweight JavaScript animation library.
- [Chart.js](https://www.chartjs.org/) - Interactive charts and graphs.
- [Loader Animations](https://uiball.com/loaders/) - CSS loader animations.
- [Driver.js](https://driverjs.com/) - Drive user focus across the page.
- [Flickity](https://flickity.metafizzy.co/) - Responsive carousel library.
- [ScrollReveal](https://scrollrevealjs.org/) - Scroll animations for web pages.

## 📚 Learning & Docs
- [Flexbox Froggy](https://flexboxfroggy.com/) - Game for learning CSS Flexbox.
- [StepZen Docs](https://us-east-a.dashboard.ibm.stepzen.com/getting-started) - GraphQL API development.
- [GraphQL PHP Schema](https://webonyx.github.io/graphql-php/schema-definition/) - Defining schemas in PHP.

## 🆓 Free Assets
- [unDraw](https://undraw.co/illustrations) - Open-source illustrations.
- [iconmonstr](https://iconmonstr.com/) - Free simple icons.
- [Google Fonts](https://fonts.google.com/) - Open-source web fonts.
