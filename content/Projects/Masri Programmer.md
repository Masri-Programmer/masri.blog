## 💻 The "Modern Monolith" Stack
The platform itself is a showcase of the **VILT stack**, our signature architectural pattern for building seamless, reactive web applications.

*   **Laravel:** The robust PHP framework for powerful, secure backend logic.
*   **Vue.js 3:** For highly interactive, progressive user interfaces.
*   **Inertia.js:** The bridge that enables a Single Page Application (SPA) feel without the complexity of a separate API.
*   **Tailwind CSS:** For rapid, utility-first custom design implementation.
